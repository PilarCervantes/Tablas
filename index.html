Atributos

Atributo Función
align Nos ayuda a alinear el texto o lo que se indique hacia la izquierda, derecha, al centro, etc.
valign Esta propiedad establece la alineación vertical de un elemento.
background Define los valores individuales del fondo de la página como pueden ser imagenes, color, etc.
bgcolor Es usado para determinar el color de fondo de un párrafo, tabla o cualquier otra parte de la página web.
bordercolor Establece el color de los cuatro bordes de un elemento.
cellpadding Permite especificar la distancia entre el margen interior de la celda y su contenido.
cellspacing Permite especificar la distancia entre las celdas y el margen exterior de la tablas.
border Agrega un borde a la tabla o a un elemento de la página.
height Se utiliza para especificar la altura de un elemento.
width Se utiliza para determinar el ancho de un elemento.
